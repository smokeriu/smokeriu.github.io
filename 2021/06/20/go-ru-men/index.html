go入门

Go
Go
发布日期:   2021-06-20

数据类型

变量定义

// 数据类型在后。与scala类似（不过不需要引号）
// 所有数据会有个初始值。int为0
var a int

// 能够自动进行类型推断。属性可省略
var a = 1

// 可以使用 := 定义变量
a := 1

// 包变量（方法外变量，类似于java的全局变量），必须使用var定义，而不能使用 :=
var aa = "a"

// 定义多个变量时。可以使用 var()
var(
    a = 1
  b = "a"
  c = "d"
)
// 也可以使用如下方式同时定义
var aa,bb,cc = 1,"2",true
aa,bb,cc := 1,"2",true

// 变量定义完后类型是确定的。go不会进行隐式数据类型转换
// 如使用math.Sqrt(x)时，x必须传入float
a, b := 1, 2
math.Sqrt(float64(a*a + b*b))

// go中，定义了的变量必须要用到

常量定义

// 使用const定义常量，类似于scala中的val，java中的final
// int可以省略
// go中首字母大写有含义，故常量不会全部大写。
const a int = 1

// 未指定int时，a能够被识别为数字数据类型
// 不需要像var一样强制转换
const a, b = 3, 4
math.Sqrt(a*a + b*b)

// go中没有枚举类型。可利用const定义枚举
func enums(){
  const (
      cpp = 1
    java = 2
  )
}

// 定义有递增关系的枚举。可使用iota
const (
    a = iota // 0
  b                 // 1
  c                 // 2
)
// 也可以使用公式
const (
  aa = 2 * (1 + iota) // 2 
  bb                  // 4
  cc                  // 6
)

数据类型

go的内建类型与java等有一定区别。

  • bool， string， byte
    • 这几类与一般语言类似
  • (u)int，(u)int8/16/32/64，uintptr
    • (u)int为不定长整形。具体范围视设备/运行的操作系统决定。带u的为无符号整型。
    • (u)int8/16/32/64为定长整型。
    • uintptr为指针
  • float32/64，complex64/128
    • float32/64为浮点数
    • complex64/128为复数。实部虚部均为float32/64
  • rune
    • 字符型。类似于java中的char。但有一些区别

条件语句

go中使用if和swith-case作为条件语句

if

if的用法和大多数语言相似。有几个区别：

  • 判断不需要加小括号。

  • 可以在if后直接跟变量定义

  • // 不需要小括号
if a == 3 {
  // do something1
}else if a == 2 {
  // do something2
}else {
  // do something3
}

// if后跟定义. a 只能在if的作用域中使用
if a := func1(); a == 1 {
  fmt.Println(a)
}else if a == 2 {
  // do something2
}else{
  // do something3
}

swith

switch用法与java类似。只是不需要在每句后break

// 可以switch某一个值
a := 1
switch a {
  case 1:
      fmt.Println("A")
  case 2:
      fmt.Println("B")
  default:
      fmt.Println("60")
}

// 可以直接在case后跟条件
a := 1
switch {
case a < 60:
    fmt.Println("A")
case a > 60:
    fmt.Println("B")
default:
    fmt.Println("60")
}

循环语句

go中只有for一种循环。与java的fori类似，分为三段。但这三段可以任意省略。

for i := 1; i < 30 ; i++ {
  // do something
}

// 省略定义i，则可以使用外部变量
i := 1
for ; i < 30 ; i++ {
  // do something
}

// 再省略递增处，则类似于一般语言的while循环
// 这种情况可以省略;号
i := 1
for i < 30 {
  // like while
}

// 省略退出条件。则是死循环
i := 1
for ; ; i++{
  fmt.Println(i)
}
// 可以全省略
i := 1
for {
  fmt.println(i)
}

函数

go的函数有几个特点：

  • func关键字
  • 返回类型在最后（类似于scala），但函数除非不需要返回值，否则必须设定返回值（不能类似于变量那样自动推断）
  • 可以返回多个返回值（写法类似于scala的元组，但必须用多个变量去接收）
    • 返回值也能够进行命名。
  • 不能使用const去接收函数的返回值？似乎是，存疑
  • 参数部分，同类型的参数可以放一起，省略类型命名
  • 对于不想要的返回值，可以使用_来接收
// 参数部分，同类型的参数可以放一起，省略类型命名
func eval(a, b int, op string) int {}
// 多个返回值
func div(a, b int) (int int){
  return 1, 2
}
// 返回值命名/合并
func div(a, b int) (q, r int){}
// 多返回值一般把第二个用来返回error

// 丢弃不需要的返回值
q, _ := div(13, 3)
fmt.Println(q)
  • 函数的参数也可以是一个函数
func apply(op func(int, int) int , a , b int) int {}
  • go不支持默认参数等。只支持可变参数列表
func sum(numbers ...int) int {}

指针

go语言的指针不能预算。只能赋值

参数传递

Go语言只有值传递。没有引用传递

即便是传递指针，也是会将指针拷贝，然后再传入函数。不过内外两个指针会指向同一个内存地址。

集合

数组

在go语言中一般不直接使用数组，而是使用切片

定义数组时，长度在前，数据类型在后。

定义数组时必须指定数据类型。利用函数返回值时例外

numbers := [5]int{1,2,3,4,5}

// 可以不定义初始值, 此时必须制定类型+长度
var numbers [5]int

// 可以由系统推断长度, 此时必须指定初始元素
numbers := [...]int{1,2,3,4,5}

// 使用[index]来取得索引上的值.go也是从0开始
a := numbers[0] // 1

[]int 是切片。[5]int才是数组

可以使用range关键字遍历数组

for index, value := range numbers {
  // do something
}

数组是值类型

  • 定义func时，如果参数时(a [5]int)，则只能接受[5]int，不能接受[3]int
  • func中修改数组内容时，其实是修改的拷贝。

切片（slice）

和python的切片很像。区间也是左闭右开

slice是array的视图

// start , end均可省略。
// slice包含start对应的内容，不包含end对应的内容。
// 若start，end均省略，则为arr数组的完整视图
slice := arr[start:end]

slice是array的视图

  • 修改slice的值，会对array本身产生影响

    slice := arr[2:]
slice[0] = 100 // arr[2] => 100

  • 当函数参数为slice时，修改slice会影响slice所指向的数组

    func update(sl []int){
    int[0] = 100
}

update(arr[2:]) // arr[2] => 100

  • 可以在slice的基础上继续slice

    • 继续slice时，可以扩展

    • s2实际上能拿到arr[6]，不过s1是没有s1[4]的。这个源于slice的实现

      • slice主要由三部分组成

        • ptr：指向slice开始的指针。

        • len：slice的长度。如果越界会报错

        • cap：从slice开始，到arr结尾的所有元素

        • s[i]不能超越len(s)，向后扩展不可以超过底层数组cap(s)

          • 能够通过len(s)，cap(3)获取slice的这两个长度。

    • slice可以向后扩展，不能向前扩展

向slice添加元素

使用append函数，append会返回一个修改后的切片

  • 首先会在len后，cap处修改元素。逐个向后修改。此时会影响底层的array
  • 如果在新的切片上增加多次，长度超过了cap，则系统会重新分配一个更大的底层数组，此时slice不再指向原本的arr

创建slice

可以直接创建slice，而不是在数组的基础上。

var s []int

s1 := []int{2, 3, 4}

// (type, len, cap)
s2 := make([]int, 16 ,32)

复制slice

使用copy方法复制slice

s1 := []int{1, 2, 3, 4, 5}
s2 := make([]int, 3, 16)
// copy(dst, src)
copy(s2, s1)
fmt.Println(s2) // [1 2 3]
s3 := s2[:4]
fmt.Println(s3) // [1 2 3 0]

删除slice的某个元素

go不自带该功能，可以使用append方法实现

s1 := []int{1, 2, 3, 4, 5}

// 删除第2个元素2。
s1 = append(s1[:1], s1[2:]...)

fmt.Println(s1) // [1 3 4 5]

移除slice首尾元素

同样不自带。可以使用切片完成

s1 := []int{1, 2, 3, 4, 5}

front := s1[0]
s1 = s1[1:]
fmt.Printf("front: %d, s1: %v\n", front, s1) // front: 1, s1: [2 3 4 5]

tail := s1[len(s1)-1]
s1 = s1[:len(s1)-1]
fmt.Printf("tail: %d, s1: %v\n", tail, s1) // tail: 5, s1: [2 3 4]

Map

Map的定义

正常Map：map[K]V,

复合Map： map[K1]map[K2]V

map是一个hashMap

// 初始化一个map
m1 := map[string]int {"id":1, "age":2}

// 空map
var m2 map[string]int // m3 == nil

m3 := make(map[string]int) // m2 == empty map

Map遍历

同样可使用range来遍历

for k, v := range m {
  // do something with k, v 
}

Map的内容获取

value := map[key]

m1 := map[string]int {"id":1, "age":2}

// ok这个参数可省略，主要用于判断key是否存在
age, ok := m1["age"] // 2, true

// 如果key不存在
other, ok2 := m1["other"] // 0 , false

当key在map中不存在时，会得到对应数据格式的zero value。

对于int，就是0。对于string，则是空字符串

Map的数据删除

使用delete函数进行删除

m1 := map[string]int {"id":1, "age":2}

// 一次只能删除一个key
delete(m1, age) // {"id": 1}

Map数据添加或修改

m := map[string]int{"id": 1, "age": 2}

m["age2"] = 2    
m["age"] = 3

fmt.Println(m) // map[age:3 age2:2 id:1]

Map的使用

  • map使用hash表，key必须能够比较
  • 除了slice, map, function的其他内建类型都可以作为key
  • struct类型不包含上述3个类型时，也可以作为key
